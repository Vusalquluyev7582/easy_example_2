HTML 5

HTML5, İnternetin əsas texnologiyalarından biri olan HTML işarələmə standartının beşinci versiyasıdır. O, qismən Internet Explorer, Mozilla Firefox və Opera brauzerlərinin cari versiyaları tərəfindən dəstəklənir. Bir sözlə, H5 eksperimental mərhələdə olsa belə, Google Chrome və Safari brauzerlərinin cari versiyalarında daha yaxşı işləyir. Bu gün istifadə edilən HTML 4.1 versiyası nə qədər müntəzəm və möhkəm şəkildə CSS dəstəyi ilə kodlaşdırılsa da, əlavə kodların hələ də onun funksionallığını pozduğu məlumdur. Buna görə HTML5 bu ehtiyacları ödəmək üçün hazırlanmağa başladı. Video paylaşma saytı YouTube sınaq mərhələsində istifadəçilərinə HTML5 təklif edir.

CSS

CSS veb səhifə yaradıcıları tərəfindən rəngləri, şriftləri, üslubları, ayrı-ayrı blokların yerləşdirilməsini və həmin veb səhifələrin görünüşünün təqdimatının digər aspektlərini müəyyən etmək üçün istifadə olunur. CSS inkişafının əsas məqsədi veb-səhifənin məntiqi strukturunun təsvirini ( HTML və ya digər işarələmə dillərindən istifadə etməklə hazırlanır) həmin veb-səhifənin görünüşünün təsvirindən (hazırda rəsmi dildən istifadə etməklə hazırlanır) hasarlamaq və ayırmaqdır. CSS. Bu ayırma sənədin əlçatanlığını artıra, onun təqdimatına daha çox çeviklik və nəzarət təmin edə, struktur məzmunda mürəkkəbliyi və təkrarı azalda bilər.

JAVASCRİPT

JavaScript çox paradiqmalı proqramlaşdırma dilidir. Obyekt yönümlü, imperativ və funksional üslubları dəstəkləyir. Bu ECMAScript spesifikasiyasının tətbiqidir (ECMA-262 standartı [2]). JavaScript adətən proqram obyektlərinə proqramlı şəkildə daxil olmaq üçün quraşdırılmış dil kimi istifadə olunur. Veb səhifələrə interaktivlik əlavə etmək üçün skript dili kimi brauzerlərdə ən çox istifadə olunur [8]. Əsas memarlıq xüsusiyyətləri: dinamik yazma, zəif yazma, avtomatik yaddaşın idarə edilməsi, prototip proqramlaşdırma, birinci dərəcəli obyektlər kimi funksiyalar.
